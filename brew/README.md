# Homebrew

## Install

```
ruby -e "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/master/install)"
```

## Usage

__Common__

```
brew install xxx 安装
brew uninstall xxx 卸载
brew search /xxx*/ 搜索
```

__Other__

```
brew list 出已安装的软件
brew update 更新brew
brew home 用浏览器打开brew的官方网站
brew info 显示软件信息
brew deps 显示包依赖
```
